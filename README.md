# duplicate

files duplicate viewer

## install

```
pip install git+https://github.com/streanger/duplicate.git
```

## usage

```bash
# from cli
duplicate

# as module
python -m duplicate
```

```python
# from python
import duplicate
duplicate.gui()
```
